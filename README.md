FastAPI backend project using async SQLAlchemy, PostgreSQL, and Docker. Includes full CRUD operations, clean project structure, and Pydantic schemas.

## Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy (Async)
- Docker & Docker Compose
- Pydantic

## Run the application
\`\`\`bash
uvicorn src.main:app --reload --port 8000
\`\`\`
